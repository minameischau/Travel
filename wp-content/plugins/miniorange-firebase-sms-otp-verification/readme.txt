== Description ==

= OTP VERIFICATION WITH FIREBASE GATEWAY =
Plugin allows endusers to login/register with one time password(OTP) with firebase gateway. Plugin uses [Firebase Gateway](https://firebase.google.com/) to send and verify OTP. It removes the possibility of users registering with fake Mobile Number by enabling OTP Verification. It allows passwordless login & login with phone number. The OTP Verification plugin also checks if Email Address/Mobile Number of a user already exists.
Follow the link for detailed information on the free and premium plan of the plugin : [MiniOrange OTP Verification using Firebase Gateway](https://plugins.miniorange.com/wordpress-otp-verification-using-firebase-gateway)

== FEATURES ==

* Login with Phone Number
* Passwordless login (No need to remember your password anymore!)
* OTP verification on registration form.
* Supports country code dropdown
* Customizable login & registration form.
* Replace the default WooCommerce Login/Registration Form with the miniOrange Login/Registration Form.
* OTP Verification on the default WooCommerce Login/Registration Form.
* OTP Verification on the WooCommerce Checkout Form.
* OTP Verification on the Ultimate Member Registration Form.
* OTP Verification on the Gravity Form.
* Allow the admin to decide the role of the user.
* Allow admin to set up custom redirection after login/ Registration.

= How does this plugin work? =
1. On submitting the registration or login form an SMS with OTP is sent to the mobile number provided by the user.
2. Once the OTP is entered, it is verified and the user gets registered.

== Installation ==

= From your WordPress dashboard =
1. Visit `Plugins > Add New`
2. Search for `miniOrange OTP verification with Firebase`. Find and Install `Miniorange OTP verification with Firebase`
3. Activate the plugin from your WordPress Plugins page.

= From WordPress.org =
1. Download miniorange-firebase-sms-otp-verification.
2. Unzip and upload the `miniorange-otp-verification` directory to your `/wp-content/plugins/` directory.
3. Activate miniOrange OTP verification with Firebase from your Plugins page.
